## AllEqual
*Description:* Check if all elements are equal

*Code:*
```
duplicate 2 2 size / - 
for { 
    max 
}

duplicate 2 2 size / - 
for { 
    min 
}

==
```

## Factorial
*Description:* Get the factorial of N

*Code:*
```
N 0 swap for {
    1 + dup
}

1 swap - for {
    *
}

```
